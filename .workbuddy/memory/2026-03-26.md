## Bug 修复 - onLoad 报错

### 问题: 添加宠物页面报错 `onLoad is not defined`
- 错误原因: 在 Vue3 组合式 API 中，`onLoad` 需要从 `@dcloudio/uni-app` 导入
- 修复方案: 
  - 将 `declare const onLoad` 改为 `import { onLoad } from '@dcloudio/uni-app'`
  - 移除空的 `onMounted` 钩子

- 构建成功 (Build complete)

## 用户信息显示优化

### 问题 1: 设置页同时显示登录入口和退出登录按钮
- **原因**: 登录状态使用 `isLoggedInRef` 未定义，退出按钮使用 `userInfo?.isLoggedIn`，两者不一致
- **修复**:
  - 添加 `isLoggedIn` 和 `currentUser` 响应式变量
  - 添加 `checkLoginStatus()` 方法从本地存储获取登录状态
  - 统一使用 `isLoggedIn` 判断显示登录卡片或退出按钮
  - 登录后显示用户邮箱（取 @ 前部分）

### 问题 2: 首页不显示用户名
- **修复**:
  - 添加 `userName` 和 `greeting` 响应式变量
  - 添加 `getGreeting()` 方法根据时间显示不同问候语（早上好/上午好/中午好/下午好/晚上好）
  - 添加 `getUserInfo()` 方法获取用户信息
  - 登录成功后更新首页用户信息

- 构建成功 (Build complete)
